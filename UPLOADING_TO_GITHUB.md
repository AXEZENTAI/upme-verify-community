# Uploading to GitHub (Fast)

1) Create a new GitHub repo (public) named:
   `upme-verify-community`

2) Unzip this package.

3) Open the extracted folder, press **Ctrl + A**, then drag everything into:
   GitHub → Add file → Upload files

You must see these at top level:
- README.md
- LICENSE
- config/
- docs/
- upme/
- examples/

Then commit with:
`Initial public release: UPME-Verify Community Edition v1.0.1`
